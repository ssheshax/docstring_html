BASIC UTILITY FUNCTIONS

These are functions that help transform data.

pipe_filter

This function is a shortcut for filter, which filters elements from an iterable based on a condition.

Example:
>>> inputages = [49, 22, 5, 12, 17, 18, 24, 9, 32]
>>> def validateadulthood(x):
...    if x < 18:
...        return False
...    else:
...        return True
>>> pip_filter = pipeline.pipe_filter(validateadulthood)
>>> result = pip_filter(inputages)
>>> print(list(result))
[49, 22, 18, 24, 32]
pipe_map

This function wraps the built-in map() function. It allows you to apply a transformation function to a list

(or iterable).

Example:
>>> pipemap_func_multiplybytwo = pipeline.pipe_map(lambda x: x * 2)
>>> list2 = [3, 6, 9]
>>> result = list(pipemap_func_multiplybytwo(list2))
>>> result
[6, 12, 18]
pipe_mapstar

This is a more advanced version of pipe_map that can handle functions expecting multiple arguments (like f(*args)).

Example:
>>> list1 = [(3, 4), (5, 6), (7, 8)]
>>> pipemapstar_func_multiply = pipeline.pipe_mapstar(lambda x, y: x * y)
>>> result = list(pipemapstar_func_multiply(list1))
>>> print(result)
[12, 30, 56]
pipe_reduce

This function wraps reduce() (from functools), which reduces an iterable to a single value using a binary function.

Example:
>>> list1 = [4, 5, 6, 7]
>>> pipereduce_func_sum = pipeline.pipe_reduce(lambda x, y: x + y, 0)
>>> result = pipereduce_func_sum(list1)
>>> print(result)
22
pipe_flatten

This function flattens a list of lists (or any iterable of iterables) into a single list.

Example:
>>> list1 = [[1, 2], [3, 4], [5, 6]]
>>> pipemap_flatten_func = pipeline.pipe_flatten
>>> result = list(pipemap_flatten_func(list1))  # Convert to list to evaluate the iterator
>>> result
[1, 2, 3, 4, 5, 6]
pipe_slice

This function slices an iterable, either extracting a single item or a range of items.

Example 1: Get the first item from the list:
>>> list1 = [10, 20, 30, 40]
>>> pipeslice_func_first = pipeline.pipe_slice(0)
>>> result = pipeslice_func_first(list1)
>>> print(result)
10
Example 2: Slice a range from the list:
>>> pipeslice_func_range = pipeline.pipe_slice(1, 3)
>>> result = pipeslice_func_range(list1)
>>> print(result)
[20, 30]
pipe_not_filter

This is a combination of pipe_filter and a negation function (pipe_not). It filters out elements that do not meet the condition.

Example:
>>> inputages = [49, 22, 5, 12, 17, 18, 24, 9, 32]
>>> def validate_adulthood(x):
...     return x >= 18
>>> pipenotfilter_func = pipeline.pipe_not_filter(validate_adulthood)
>>> result = list(pipenotfilter_func(inputages))
>>> print(result)
[5, 12, 17, 9]

CORE/PRIMITIVE UTILITY FUNCTIONS

These are functions that make using pipelines easier and simpler.

partial2

Takes a function and an argument that will return a new function. The data in the pipeline will be the FIRST argument and the argument given initially will be the SECOND argument to the function given earlier.

Example:
>>> def multiply(a, b):
...     return a * b
>>> multiply_by_5 = pipeline.partial2(multiply, 5)
>>> result = multiply_by_5(10)
>>> print(result)
50
partial2star

Takes a function and variable number of arguments that will return a new function. The data in the pipeline will be the FIRST argument and the arguments given initially will be the rest of arguments to the function given earlier.

Example:
>>> def aggregatefunc(arg1, *args):
...     result = 0
...     for arg in args:
...         result += arg
...     return arg1 + " " + str(result)
>>> concatstring = pipeline.partial2star(aggregatefunc, 2, 3, 4, 5)
>>> output = concatstring("The result is")
>>> print(output)
The result is 14
partial2kwstar

Example of using partial2kwstar to pass keyword arguments to a function.

Example:
>>> def greet(name, greeting="Hello"):
...     return f"{greeting}, {name}!"
>>> greetings = pipeline.partial2kwstar(greet, {"greeting": "Hi"})
>>> result = greetings("John")
>>> print(result)
Hi, John!
pipe_apply

Takes variable number of arguments that will return a new function which will pass on the initially given arguments to the function in the pipeline.

Example:
>>> def check_even(num):
...     result = num % 2 == 0
...     return result
>>> iseven = pipeline.pipe_apply(check_even)
>>> output = iseven(4)
>>> print(output)
True
pipe_applystar

Takes variable number of arguments that will return a new function which

will pass on the initially given arguments to the function in the pipeline.

Example:
>>> def aggrfuncion(*args):

...     for arg in args:
...         result += arg
...     return result
>>> addintegers = pipeline.pipe_applystar(aggrfuncion)
>>> output = addintegers(3, 5, 7)
>>> print(output)

pipe_applykws

Example of passing keyword arguments using pipe_applykws.

Example:

class libs.core.pipeline.Pipeline(**options: Optional)

Bases: object

A class that implements a functional pipeline allowing chaining of functions in a readable way. This pipeline allows you to define a sequence of transformations (functions) and apply them to an initial value. Functions can be added to the pipeline using the >> operator, and the pipeline can be executed with a call to the instance itself.

Example

>>> list1 = [1, 2, 3, 4]
>>> res = Pipeline() >> pipeline.pipe_map(lambda x: x * 2) >> list
>>> output = res(list1)
>>> print(output)
[2, 4, 6, 8]

The pipeline supports optional decorators and provides the ability to exit early based on specific conditions.
